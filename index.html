คู่มือการใช้งานระบบตรวจสอบยอดชำระฌาปนกิจ

Version 1.0 | สำหรับเจ้าหน้าที่ผู้ดูแลระบบ

1. ภาพรวมของระบบ

ระบบนี้ออกแบบมาเพื่อช่วยอำนวยความสะดวกในการตรวจสอบสถานะการชำระเงินของสมาชิก (สสอ.รท และ สส.ชสอ) โดยการนำเข้าไฟล์ข้อมูล Excel หรือ CSV ระบบจะทำการประมวลผลและแสดงยอดสรุปแยกตามสังกัด รวมถึงระบุสถานะรายบุคคลว่า "ชำระเงินแล้ว" หรือ "รอชำระ"

2. การเริ่มต้นใช้งานและการนำเข้าข้อมูล

เมื่อเข้าสู่หน้าแรกของระบบ ท่านจะพบกับหน้าจอเริ่มต้นที่พร้อมสำหรับการอัปโหลดไฟล์

ขั้นตอนการปฏิบัติ:

  1. เลือกไฟล์: คลิกที่ปุ่ม Choose File บริเวณมุมบนซ้าย
  2. เลือกไฟล์ข้อมูล: เลือกไฟล์นามสกุล .xlsx หรือ .csv จากเครื่องคอมพิวเตอร์ของท่าน
  3. ตรวจสอบ: คลิกปุ่มสีเขียว ⚡ ตรวจสอบ เพื่อให้ระบบประมวลผล
  4. ล้างข้อมูล: หากต้องการยกเลิกหรือเริ่มใหม่ ให้กดปุ่มสีเหลือง ↺ ล้างข้อมูล

3. การตรวจสอบสมาชิกที่ "ชำระเงินแล้ว"

เมื่อข้อมูลถูกโหลดเข้าสู่ระบบ ท่านสามารถกรองดูเฉพาะผู้ที่ชำระเงินแล้วได้ โดยคลิกที่ตัวเลือกสถานะ "ชำระเงินแล้ว" ด้านซ้ายบน

ข้อสังเกตข้อมูล:

  • ยอดที่ชำระ: จะแสดงเป็น ตัวเลขสีเขียว ตามจำนวนที่จ่ายจริง
  • สถานะสังกัด: แสดงเครื่องหมายถูกสีเขียว (✅) ในช่อง สสอ.รท หรือ สส.ชสอ
  • สถานะ: ป้ายกำกับจะเป็นสีเขียวเขียนว่า ชำระเงินแล้ว

4. การตรวจสอบสมาชิกที่ "รอชำระ"

หากต้องการติดตามยอดค้างชำระ ให้เลือกสถานะเป็น "รอชำระ" ระบบจะแสดงรายชื่อสมาชิกที่ยังไม่ได้ชำระเงิน

ข้อสังเกตข้อมูล:

  • ยอดที่ชำระ: จะแสดงเครื่องหมายขีด (-) หมายถึงยังไม่มียอดเงินเข้า
  • สถานะสังกัด: แสดงเครื่องหมายกากบาทสีแดง (❌) ระบุว่ายังไม่ผ่านเกณฑ์การชำระ
  • สถานะ: ป้ายกำกับจะเป็นสีเทาเขียนว่า รอชำระ

5. การอ่านค่า Dashboard และรายละเอียดเชิงลึก

ระบบจะสรุปยอดรวมไว้ที่ Dashboard ด้านบนขวา เพื่อให้ทราบจำนวนผู้ชำระเงินแล้วในแต่ละสังกัดทันที

ส่วนประกอบ คำอธิบาย
กรอบสรุปยอด (Dashboard) แสดงจำนวนสมาชิกที่ชำระแล้ว แบ่งเป็น "สสอ.รท" และ "สส.ชสอ" (ตัวเลขสีเขียว)
Export CSV ปุ่มสำหรับดาวน์โหลดข้อมูลที่แสดงผลอยู่ในตารางออกมาเป็นไฟล์ Excel/CSV เพื่อนำไปใช้งานต่อ
ช่องทาง ระบุวิธีการชำระเงิน เช่น "เงินสด" หรือ "เงินโอน"
วันที่ ระบุวันที่ที่มีการชำระเงินเข้ามา (วว/ดด/ปปปป)
